Quick Start Guide

Get TrackForcePro installed and ready in minutes

Version 1.8.0
Install from Chrome Web Store

1 Install the Extension

Chrome Web Store (Recommended)

The easiest way to get started — one click to install:

Add to Chrome — It's Free

Also available for: Microsoft Edge, Firefox, and Opera. Visit the respective app stores for your browser.

For Developers (Manual Installation)

If you want to load the extension from source:

Enable Developer Mode

In Chrome, go to chrome://extensions and toggle "Developer mode" (top-right)

Load the Extension

Click "Load unpacked," navigate to the TrackForcePro folder, and select it

Pin the Extension

Click the puzzle piece icon in your toolbar and pin TrackForcePro for quick access

2 Open TrackForcePro

Once installed, click the TrackForcePro icon in your browser toolbar to open the extension popup.

The Main Interface

🧭 Tabs

Access 9 major tabs: Audit Trail, SOQL Builder, GraphQL Builder, Data Explorer, Platform Events, LMS, and Settings

🔌 Connection Status

Green = Connected to Salesforce. Red = Not connected (need to log in)

⧉ Pop Button

Click to open as a tab. Shift+click to open as a standalone window

Screenshot: Main Extension Popup
Shows the tabbed interface with Audit Trail, SOQL, GraphQL, Data Explorer tabs and connection status indicator

3 Connect to Salesforce

TrackForcePro automatically detects your Salesforce session from browser tabs.

Open Salesforce

In another browser tab, go to your Salesforce org and log in

Keep It Open

Don't close the Salesforce tab — the extension needs it to stay active

Check Connection Status

Open the extension popup. You should see a green connection indicator showing you're connected

Connection Troubleshooting

StatusMeaningAction
🟢 GreenConnectedYou're ready to use all features!
🔴 RedNot ConnectedLog into Salesforce in another tab
TrackForcePro requires an active Salesforce tab with a valid login. Keep that tab open while using the extension.

4 Explore the Features

🔍 Audit Trail Tab

Extract, search, and export Salesforce Setup Audit Trail logs. Filter by date, user, object, or action. Export as CSV or JSON.

📊 SOQL Builder

Visually build and execute SOQL queries. Select objects, fields, add filters, ordering, and limits. See results instantly with query history.

🔗 GraphQL Builder

Advanced query builder with schema autocomplete, syntax highlighting, and support for child records. For experienced users.

💾 Data Explorer (6 Tools)

🔍 Record Scanner

Scan records by ID, view all fields, edit values, navigate related records

👤 User Manager

Search users, view/edit details, manage permissions, bulk operations

🔒 Security Manager

Browse profiles, permission sets, permission set groups

🏢 Org Tools

Login As users, manage labels, deployments, scheduled jobs

🎨 Org Manager

Customize org branding, manage favicons, sandbox labels

📡 Platform Events & 📢 LMS

Subscribe to and monitor Platform Events and Lightning Message Service channels in real-time.

5 Use Show All Data on Record Pages

When viewing any Salesforce record detail page, you'll see a floating TF button at the bottom-right corner.

Open a Record

Navigate to any Account, Contact, Opportunity, or custom object record in Salesforce

Click the TF Button

Look for the floating TF button at the bottom-right. Click it to open the All Fields inspector

Explore All Fields

See every field on the record (hidden and visible). Search, filter, double-click to edit, and navigate lookups

You can also use the keyboard shortcut Ctrl+Shift+X to toggle Show All Data (customizable in Settings).

6 Access the TF Sidebar

The TF Sidebar gives you quick access to tools without leaving the Salesforce page.

Open the Sidebar

Press Ctrl+Shift+S (or your custom shortcut) on any Salesforce page

Quick Tools

The sidebar includes tools like Login As, Language Switcher, and Cache Clearer at the top

Tool Shortcuts

Click any shortcut to open that tool in the main extension popup

Screenshot: TF Sidebar
Shows the floating sidebar panel with quick tools and shortcuts to all major features

7 Customize Your Experience

Open the extension and click the Settings tab to customize TrackForcePro.

General Settings

🌓 Theme

Switch between Light and Dark theme. Your preference is saved automatically

⌨️ Keyboard Shortcuts

Customize shortcuts for Toggle Sidebar, Show All Data, and Command Palette. Click any action to rebind

🔐 Production Safety

"Disable Inline Edit in Production" prevents accidental field changes in production orgs

Your customizations are saved to your browser automatically and persist across sessions.

⚡ Pro Tips

Keyboard Shortcuts Save Time

  • Ctrl+K opens the Command Palette — search any feature
  • Ctrl+Shift+S toggles the sidebar on Salesforce pages
  • Ctrl+Enter executes queries in builders

Inline Editing is Powerful

  • Double-click any field in Data Explorer to edit directly
  • Changes save instantly to Salesforce via the API
  • Disable editing in production if you're nervous — see Settings

Data Explorer is Your Swiss Army Knife

  • Use Record Scanner to explore any record by ID
  • Navigate related records via lookup fields with breadcrumbs
  • Pin frequently-accessed records for quick re-lookup

Keep Your Session Alive

  • Always keep your Salesforce tab open — the extension depends on it
  • If you see "Not Connected," refresh the Salesforce page
  • Long-running sessions may expire — re-login if needed
