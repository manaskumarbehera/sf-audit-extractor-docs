Quick Start Guide

Get TrackForcePro installed and ready in minutes. Premium toolkit for Salesforce admin and developer workflows.

Version 2.1.6

1 Install the Extension

One-Click Installation

TrackForcePro is available on all major browser extension stores. Choose your browser and click to install:

🌐 Chrome Web Store 🔷 Edge Add-ons 🦊 Firefox Add-ons ⭕ Opera Add-ons

Click "Add to Chrome" (or your browser)

You'll see a permissions dialog. Review and confirm to install.

Pin the extension to your toolbar

Click the Extensions menu in your browser, find TrackForcePro, and pin it for quick access.

You're done

The extension is ready to connect to Salesforce. No additional setup required.

Tip: On Firefox, the extension will show a notification the first time you visit Salesforce confirming that it's active.

2 Connect to Salesforce

Automatic Session Detection

TrackForcePro requires no separate login. It detects your Salesforce session automatically using browser cookies.

Navigate to your Salesforce org

Open any Salesforce org (prod, sandbox, scratch org, etc.) in your browser.

Watch for the connection indicator

The TrackForcePro extension icon will show a green dot when your session is detected. This means the toolkit is ready to use.

You're connected

If the indicator is gray, refresh the Salesforce page or check that you're logged in. The extension will auto-detect once authenticated.

Security note: TrackForcePro uses your existing Salesforce session cookies for authentication. It never stores your password, and all API calls use your org's access token.

3 Choose Your Surface

Three Ways to Interact with TrackForcePro

The extension works across three surfaces. Choose based on your task:

🚀 Launcher

Click the TrackForcePro icon in your toolbar for quick actions: query history, pinned records, sidebar toggle, and command palette. Perfect for fast context switching.

💼 Workspace

Full-featured tab in your browser with all tools: SOQL/GraphQL builders, record explorer, audit trails, org health, platform events, and debug logs. Your primary workspace.

⚡ In-Page

Keyboard shortcuts and overlays on Salesforce pages: Ctrl+Shift+S for sidebar, Ctrl+K for command palette, or double-click record fields to inspect all data.

Quick Reference

Surface When to Use Access
Launcher Quick actions, history, search Click extension icon
Workspace Deep work, complex queries, analysis Click "Open Workspace" or pin as tab

4 Explore the Modes

Three Workflow Modes

The Workspace tab uses a 3-mode navigation model. Switch modes to focus on your current task:

Explore

🔵 Explore

For query and record exploration. Includes SOQL builder, GraphQL builder, REST explorer, Records (data explorer with inline edit), Schema browser, Users & Security, and field/object search.

Debug

🟠 Debug

For monitoring and troubleshooting. Platform Event streaming (LMS Explorer), debug logs (real-time capture & analysis), and network trace inspection. Watch events flow and debug issues live.

Analyze

🟢 Analyze

For audits and health checks. Audit Trail (login history, config changes), Org Health (limits, metadata usage), Admin checks, and Org Identity (instance, org ID, feature flags).

Click the mode pill at the top of the Workspace to switch. Each mode has its own set of tabs and features tailored to that workflow.

5 Try These First

Five Quick Wins to Get Comfortable

Here are the best starting points for learning TrackForcePro:

📝

Run Your First Query

Open Workspace → Explore → SOQL Builder. Type a simple query like SELECT Name FROM Account LIMIT 5 and click Execute. See results instantly.

🔍

Inspect a Record

Go to any Salesforce record. Press Ctrl+Shift+S to open the sidebar, or click the "Show All Data" button. See all fields, values, and edit inline without leaving the page.

⚡

Open Command Palette

Press Ctrl+K on any Salesforce page. Type a command name like "SOQL" or "Audit" to jump to any feature. Great for power users.

📊

Check Org Health

Open Workspace → Analyze → Org Health. See your org's limits, storage usage, API call counts, and feature flags at a glance.

🔐

Audit Login History

Open Workspace → Analyze → Audit Trail. View login history, config changes, and data modifications with timestamps and user info.

⚙️

Customize Settings

Open Workspace → Settings. Toggle themes, adjust sidebar position, configure per-org inline edit permissions, and set custom keyboard shortcuts.
