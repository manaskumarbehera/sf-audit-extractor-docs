1 Installation

Option 1: Install from Chrome Web Store (Recommended)

Easiest Method! Install directly from the official Chrome Web Store with one click:

Add to Chrome — It's Free

Option 2: Manual Installation (Developer Mode)

For developers or if you want to use the latest development version:

Download the Extension

Download the TrackForcePro extension package (.zip file) from the official source.

Extract the Files

Right-click the downloaded .zip file and select "Extract All" (Windows) or double-click to extract (Mac).

Open Chrome Extensions Page

Open Google Chrome and navigate to chrome://extensions/

Screenshot: Chrome Extensions Page
Shows the chrome://extensions/ page with extension cards

Enable Developer Mode

Toggle the "Developer mode" switch in the top-right corner of the extensions page.

Screenshot: Developer Mode Toggle
Shows the Developer mode toggle switch enabled (blue)

Load the Extension

Click "Load unpacked" button, navigate to the extracted TrackForcePro folder, and select it.

Pin the Extension

Click the puzzle piece icon in Chrome's toolbar and pin TrackForcePro for easy access.

After installation, navigate to any Salesforce page — you'll see a floating TF button at the bottom-right corner of the page. Click it to access all TrackForcePro tools instantly!

2 First Launch

On any Salesforce page, look for the floating TF button at the bottom-right corner of the screen. Click it to open the TF Sidebar — your launchpad to all TrackForcePro tools.

Screenshot: TF Sidebar on Salesforce Page
Shows the floating TF button at bottom-right and the expanded sidebar panel with tool shortcuts

Interface Overview

🧭 TF Sidebar

Floating button at bottom-right of every Salesforce page. Click to access all tools without leaving the page.

📑 Tab Navigation

Full popup with tabs: Audit Trails, GraphQL Builder, SOQL Builder, Data Explorer, Platform Events, Settings

Shows whether you're connected to a Salesforce org (green = connected, red = not connected)

Open TrackForcePro as a tab (click) or window (Shift+click)

Quick Tools (Built into TF Sidebar)

The sidebar includes built-in quick tools at the top — use them without leaving the current page:

🔓 Login As

Search for a user and log in as them. Three modes: current tab, new window, or incognito (preserves your admin session)

🌐 Language Switcher

Instantly change the current user's Salesforce language

🗑️ Clear Cache

Clear browser cache, localStorage, sessionStorage and hard refresh

3 Connecting to Salesforce

TrackForcePro automatically detects your Salesforce session from open browser tabs.

Open Salesforce

Open a new browser tab and navigate to your Salesforce org. Log in if needed.

Keep the Tab Open

Keep the Salesforce tab open in your browser (don't close it).

Open TrackForcePro

Click the floating TF button at the bottom-right corner. The extension will automatically detect your session.

Connection Status Indicators

IndicatorStatusWhat to Do
🟢 GreenConnectedYou're ready to use all features!
🔴 Red / "Not Connected"DisconnectedOpen a Salesforce tab and log in
The extension requires an active Salesforce tab with a valid login session. If you see "Not Connected," ensure you're logged into Salesforce in another tab.

7 Troubleshooting

Common Issues

❌ "Not Connected" Error

  • Open a new tab and log into Salesforce
  • Make sure the Salesforce tab stays open
  • Refresh the Salesforce page
  • Close and reopen the extension

❌ Objects Not Loading

  • Check your internet connection
  • Verify you're logged into Salesforce
  • Click "Refresh Objects" button
  • Check the browser console for errors (F12 → Console)

Error Messages

ErrorMeaningSolution
"No access token"Session expiredLog into Salesforce again
"Invalid ID length"Wrong ID formatUse 15 or 18 character ID
"Handshake failed"Streaming connection issueRefresh and reconnect
