Quick Start Guide

Get TrackForcePro installed and ready in minutes

Version 1.9.4
What's new: LWC Explorer now separates Apex, UI API, GraphQL, and Fetch/API traffic, and both edit actions open the large in-page editor window.

1 Install the Extension

Install from Chrome or Firefox

TrackForcePro is available on the Chrome Web Store and on Firefox Add-ons. Edge stays a placeholder in the docs until that store listing is live.

Choose your browser from the availability row below. Chrome and Firefox are available now, and Edge will appear here once the listing is live.

Chrome Web Store Primary install path Available now Firefox Add-ons Mozilla listing Available now
Edge Add-ons Listing coming soon Coming soon

Visit the Chrome Web Store

Search for "TrackForcePro" or use the direct link above to go straight to the listing.

Click "Add to Browser"

Accept the required permissions. TrackForcePro needs access to Salesforce domains to function — it never collects or sends data to external servers.

Pin the Extension

Click the puzzle piece (extensions) icon in your toolbar and pin TrackForcePro for quick one-click access.

Trainer note: Chrome and Firefox are the current direct-install paths. Keep Edge labeled as a placeholder in training materials until the Edge Add-ons listing is actually published.

2 Connect to Salesforce

TrackForcePro automatically detects your Salesforce session from open browser tabs. No extra login required.

Log in to Salesforce

Open a browser tab, navigate to your Salesforce org, and log in as you normally would

Keep the Salesforce Tab Open

The extension reads your session from the open tab — don't close it while using TrackForcePro

Open TrackForcePro

Click the TrackForcePro icon in your browser toolbar. A green indicator means you're connected and ready

StatusMeaningAction
🟢 GreenConnected to SalesforceYou're ready to use all features
🔴 RedNo Salesforce session detectedLog into Salesforce in another tab and refresh
Important: TrackForcePro requires an active, logged-in Salesforce tab. If your session expires, log in again and the extension will reconnect automatically.

3 Explore the Main Interface

TrackForcePro works across 3 surfaces: a launcher popup for quick entry, a workspace for deep tasks, and in-page overlays for contextual work. The workspace includes the core feature areas below.

🔍 Audit Trails

Fetch, filter, and export Setup Audit Trail logs for compliance and change tracking

📊 SOQL Builder

Visual query builder with searchable objects, field suggestions, multi-tab editing, and 5 export formats

🔗 GraphQL Builder

Three-screen workflow with CodeMirror editor, autocomplete, draggable split panes, and export

🔬 Records

Look up records by ID, view all fields, edit inline, navigate lookups, pin favourites, bulk scan 200 IDs

🏢 Org Tools

7 panels: Login As, Custom Labels, Deployments, Scheduled Jobs, Debug Logs, Org Limits, Admin Utils

🎨 Org & Favicon

Design custom favicons per org to visually distinguish Production, Sandbox, UAT, and Dev environments

📡 Platform Events

Subscribe to and monitor Platform Events in real-time with CometD streaming

💬 LMS

Subscribe to and publish Lightning Message Service channels for component testing

⚙️ Settings

Configure org management, appearance, general controls, sidebar behavior, layout, backup, and cross-surface apply behavior

❓ Help

Documentation links, quick tips, and troubleshooting guides

🗂️ Setup Search

Search Setup pages and metadata by API name directly from Salesforce pages, then jump into the workspace when the task gets deeper

🧩 LWC Explorer

Inspect the component tree on Salesforce pages, review cleaner request traces by source type, and open LWC or Apex code in the full editor window

Within Records, you also get sub-tabs for User Manager, Security Manager, and advanced metadata tools.

Quick navigation: TrackForcePro gives you two Command Palettes: the launcher/workspace Command Palette for cross-feature navigation and the Command Palette (Page) for page-aware Salesforce actions. Use Setup Search (Page) separately when you want Setup navigation plus metadata search by API name.

4 Use Show All Data on Record Pages

When viewing any Salesforce record detail page, you'll see a floating TFP button on the page edge. This works directly on the page — no popup needed. You can change the button position (Right, Left, or Bottom) in Settings > Sidebar.

Open Any Record

Navigate to any Account, Contact, Opportunity, Case, or custom object record in Salesforce

Click the TFP Button (or use your assigned shortcut)

The All Fields inspector opens as a side panel — showing every field on the record, including hidden and system fields

Search, Edit, Navigate

Search fields by name, double-click to edit values inline, click lookup fields to jump to related records with breadcrumb navigation

Keyboard shortcut: Show All Data can be assigned in Settings > General. It is best treated as an optional page-action shortcut rather than a built-in default.

5 Access the TF Sidebar

The TF Sidebar gives you quick access to tools on every Salesforce page without leaving Salesforce first.

Open the Sidebar

Press Ctrl+Shift+S (or your custom shortcut) on any Salesforce page

Quick Tools at the Top

Language Switcher, Cache Clearer, Login As, and other quick-action tools appear at the top of the sidebar

Feature Shortcuts Below

Click any shortcut to open that tool in the workspace or launcher flow. Categories include Records tools and Org Tools, while page overlays remain best for context-first tasks.

Stays visible: The sidebar persists through Salesforce SPA navigation — it remains open as you move between pages.

⚡ Pro Tips

Keyboard Shortcuts Save Time

  • Open the popup Command Palette from the header trigger or your saved shortcut. In popup mode, browsers may intercept Ctrl/Cmd+K.
  • Ctrl+Shift+S toggles the sidebar on Salesforce pages
  • Assign a shortcut for Show All Data in Settings > General if you want page-level keyboard access.
  • Use Command Palette (Page) and Setup Search (Page) on Salesforce pages when page context matters
  • Ctrl+Enter executes queries in SOQL and GraphQL builders

Inline Editing is Powerful — and Safe

  • Double-click any field in Records, SOQL results, or Show All Data to edit directly
  • Changes save instantly to Salesforce via the REST API
  • Disable editing per org in Settings > Org Management to protect production data

Org & Favicon Differentiates Your Environments

  • Assign distinct colours and shapes to Production vs. Sandbox
  • Custom favicons update all open tabs for that org instantly
  • Never accidentally work in the wrong environment again

Keep Your Session Alive

  • Always keep your Salesforce tab open — the extension depends on it
  • If you see "Not Connected," refresh the Salesforce page or log in again
  • The extension automatically recovers from service worker crashes using cookie fallback
