Quick start in under 10 minutes

Get TrackForcePro installed, connected, and running your first Salesforce productivity workflow fast.

Checklist

  • Install extension.
  • Open Salesforce tab.
  • Launch TF sidebar.
  • Run first query.

Step-by-step guide

1) Install
Use Chrome Web Store and pin if you prefer toolbar access.
2) Connect
Log in to Salesforce in any browser tab.
3) Launch
Click the floating TF button on the page.
4) Query
Open SOQL Builder and run a starter query.
5) Explore
Try GraphQL Builder, Record Scanner, and Security Manager.
6) Safety setup
Configure favicon color/label per org.

Starter query

SELECT Id, Name FROM Account LIMIT 10

Use this query in SOQL Builder to validate session and permissions.
