📌 Sidebar Guide

Quick-access floating toolbar on Salesforce pages with shortcuts to TrackForcePro tools.

In-Page Overlay

What It Does

The TrackForcePro sidebar is a floating toolbar that appears on Salesforce pages, giving you one-click access to your most-used TrackForcePro tools. Instead of opening the extension popup every time you need to run a SOQL query or check a record's fields, the sidebar keeps your favorite shortcuts visible right on the page.

The sidebar is persistent and intelligent — it stays visible as you navigate through Salesforce, and you can customize which shortcuts appear so you see only the tools you actually use. It acts as a persistent launchpad for admin and developer workflows without cluttering the Salesforce interface.

How to Open

The sidebar appears automatically on Salesforce pages. Look for the floating vertical toolbar on the right side of the page (or left/bottom if you've configured it).

You can toggle the sidebar's visibility with a keyboard shortcut (configurable in Settings). Press the shortcut to show or hide the sidebar instantly without reaching for the mouse. This is perfect if you need to reclaim screen real estate temporarily.

The sidebar position is independent — you can place it on the right, left, or as a bottom toolbar. Configure the position in Settings → Sidebar. If Salesforce's native sidebar is on the left, put TrackForcePro's on the right to avoid overlapping.

Default Shortcuts

The sidebar comes with built-in shortcuts to the most common TrackForcePro features:

Shortcut Action Category
Show All Data Opens the all-fields inspector for the current record. See every field, value, type, and editability status instantly. rec-tool
LWC Explorer Launches the component and network trace inspector on the current Salesforce page. Inspect Lightning Web Components, Aura, and LMS channels. rec-tool
SOQL Builder Switches to the SOQL Builder tab in the TrackForcePro workspace. Write, test, and execute SOQL queries instantly. rec-tool
Records Opens the Records (formerly Data Explorer) tab in the workspace. Lookup and inspect any Salesforce record. rec-tool
Org Tools Switches to the Org Tools tab. Access metadata, limits, custom settings, and other org-level utilities. org-tool
Setup Search Opens Salesforce Setup with a search field pre-focused. Search for pages, objects, fields, and settings by name. org-tool
Copy Record ID Copies the current record's 18-character ID to your clipboard. Useful for SOQL queries and API calls. org-tool

Customizing Shortcuts

Not every shortcut is useful for every user. TrackForcePro lets you customize the sidebar by showing and hiding individual shortcuts.

To customize:

  1. Open the TrackForcePro workspace (click the extension icon).
  2. Navigate to the Settings tab.
  3. Scroll to the "Sidebar" section under General settings.
  4. You'll see a list of all available shortcuts grouped by category:
  • rec-tool: Records-related tools (Show All Data, LWC Explorer, SOQL Builder, Records, etc.)
  • org-tool: Org-level tools (Org Tools, Setup Search, Copy Record ID)

Each shortcut has a toggle switch. Turn off the switch to hide that shortcut from the sidebar. Turn it back on to re-enable it. Your changes are saved immediately.

Pro Tip: If you rarely use Setup Search or Copy Record ID, hide them to keep the sidebar compact. Show only the 2–3 tools you use daily.

Position Settings

TrackForcePro gives you independent control over the position of three elements:

  1. Sidebar: The main floating toolbar with shortcuts.
  2. Show All Data Button: The floating button that opens the all-fields inspector.
  3. Command Palette Trigger: The button that opens the command palette.

Each has three position options:

  • Right: Floats on the right edge of the Salesforce page.
  • Left: Floats on the left edge (opposite of Salesforce's native sidebar).
  • Bottom: Renders as horizontal buttons anchored to the bottom edge, perfect for touchscreen use or compact layouts.

To configure positions: Open Settings → Sidebar and use the dropdown menus for each element. Your preference is saved to storage.

💡 Layout Strategy: If Salesforce's native sidebar occupies the left, place the TrackForcePro sidebar on the right. Put Show All Data on the opposite corner from the sidebar to avoid crowding. Use bottom position for Command Palette on touch-heavy workflows.

Tips & Tricks

💡 Tip 1: Keep It Minimal
Hide shortcuts you never use. A slim sidebar with 2–3 tools is more usable than a toolbar crammed with options. Enable only what you actually click.
💡 Tip 2: Position for Non-Overlap
Place the sidebar on the opposite side from Salesforce's native sidebar. If native is left, put TrackForcePro on the right. This prevents overlapping UI and makes both sidebars immediately accessible.
💡 Tip 3: Use Keyboard Toggle
Set a keyboard shortcut to toggle the sidebar visibility in Settings → Sidebar. Pressing a key to hide the sidebar is faster than reaching for it when you need screen space.
💡 Tip 4: Independent Button Positioning
Remember that Show All Data and Command Palette trigger buttons have separate position settings. You can have the sidebar on the right, Show All Data on the left, and Command Palette on the bottom — all at the same time, no conflicts.
💡 Tip 5: Bottom Layout for Mobile
If you use Salesforce on tablet or touch devices, try the "Bottom" position for all three elements. Buttons at the bottom edge are easier to reach with your thumb without blocking content.
💡 Tip 6: Sidebar Doesn't Block Form Save
The sidebar is overlay-only and won't interfere with form submission, page navigation, or any Salesforce functionality. It's completely non-blocking.
