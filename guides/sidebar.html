📌 Sidebar Guide

What It Does

The TrackForcePro sidebar is a floating toolbar that appears on the right (or left) side of every Salesforce page. It gives you one-click access to quick-access shortcuts for your most-used features: Records, Org Tools, SOQL Builder, REST Explorer, Platform Events, and more. Think of it as your personal toolbar that brings your favorite admin/dev tools within arm's reach.

Why Use It

The sidebar keeps your workflow focused:

  • Always Visible — Sidebar stays on every Salesforce page you visit. No hunting through extension UI.
  • One-Click Access — Launch your most-used tools without opening the main extension popup.
  • Contextual Shortcuts — Shortcuts adapt based on the page you're on. On a record page? See Records, Show All Data, and Field History shortcuts.
  • Keyboard Shortcut — Assign a custom keyboard shortcut to toggle the sidebar on/off (default: Ctrl+Shift+S).
  • Configurable — Show only the shortcuts you use. Hide the ones you don't.
  • Resizable — Drag the left edge of the sidebar to adjust its width to your preference.

How to Access

The sidebar appears automatically when you visit any Salesforce page. It's a floating panel on the right edge (or left, if you've configured it that way). The sidebar contains:

  • Sidebar Header — Shows "TrackForcePro" branding and a close button (X)
  • Quick Shortcuts — List of clickable shortcuts grouped by category
  • Collapse/Expand — Toggle sections to hide/show specific shortcut groups

Show/Hide the Sidebar

  • Click the X button in the sidebar header to collapse it
  • Use your keyboard shortcut (default: Ctrl+Shift+S on Windows, Cmd+Shift+S on Mac)
  • Command Palette — Type "Toggle Sidebar" in the Command Palette and press Enter

Sidebar Features

Floating Panel Design

  • Always on Top — Sidebar floats above all page content. Drag the handle to reposition if needed.
  • Light/Dark Theme — Sidebar automatically matches your TrackForcePro theme setting.
  • Responsive — On mobile or narrow screens, the sidebar adapts its layout to stay usable.

Shortcut Categories

Shortcuts are organized into collapsible sections based on feature area:

  • Explore — Records, Schema, Users, Security tools
  • Query — SOQL Builder, GraphQL Builder, REST Explorer
  • Debug — Platform Events, Debug Logs, LWC Explorer
  • Audit & Analyze — Audit Trail, Org Tools, Org Limits
  • On This Page — Context-specific shortcuts for the current page (e.g., Show All Data on record pages)

Adaptive Shortcuts

The sidebar is smart about context. For example:

  • On a record page — Show All Data, Field History, and Sharing shortcuts appear
  • On a list view — Bulk record operations and list export shortcuts appear
  • On setup pages — Metadata Explorer and org configuration shortcuts appear

Customization

Control Which Shortcuts Appear

  1. Open the TrackForcePro popup and go to Settings
  2. Find the Sidebar Settings section
  3. Toggle each shortcut category on/off
  4. Click Save & Apply

Result: Disabled shortcuts don't appear in the sidebar, but are still accessible from the Command Palette if you need them.

Resize the Sidebar

  1. Hover over the left edge of the sidebar (you'll see a resize cursor)
  2. Click and drag left to make the sidebar narrower, right to make it wider
  3. The sidebar width is saved (range: 500px to 900px)

Change Sidebar Position

  1. Go to Settings → Sidebar Settings
  2. Choose Right (default) or Left
  3. Click Save & Apply
Pro Customization: Disable shortcuts you don't use → Adjust width to 550px for compact layout → Position sidebar on your preferred side. Your ideal sidebar takes 2 minutes to set up.

Keyboard Shortcuts

Toggle Sidebar

  • Windows: Ctrl+Shift+S (default)
  • Mac: Cmd+Shift+S (default)

You can customize this shortcut in Settings → Keyboard Shortcuts → "Toggle Sidebar".

Navigate Shortcuts

  • Click any shortcut to open that tool in the extension popup
  • Right-click a shortcut for context menu options (coming soon)

Tips & Best Practices

  • Minimize visual clutter — Disable shortcuts you never use. A cleaner sidebar means faster decision-making.
  • Use the toggle shortcut frequently — Memorize Ctrl+Shift+S (or your custom shortcut) and toggle the sidebar on/off to maximize screen real estate when needed.
  • Keep sidebar width consistent — Set your preferred width once, then don't resize it. Consistency helps muscle memory.
  • Check context-aware shortcuts — The sidebar shows different shortcuts based on your current page. On record pages, Show All Data and Field History appear automatically.
  • Pair with keyboard shortcuts — Assign keyboard shortcuts to your most-used tools (e.g., Ctrl+Shift+A for Show All Data). Then you can access them via keyboard instead of clicking the sidebar.
  • Right side is default, left is personal preference — If you prefer left-side tools, move the sidebar to the left in Settings. No right/wrong choice.
Power User Workflow: Use Ctrl+Shift+S to toggle sidebar on/off as you work. When you need a tool, click the sidebar shortcut or use its keyboard shortcut. The sidebar stays out of the way until you need it.
