⌨️ Command Palette Guide

Search commands, jump to tabs, and look up records from anywhere in TrackForcePro using Ctrl+K.

Global Feature

What It Does

The Command Palette is a global search-and-command interface that lets you quickly access any TrackForcePro feature without clicking through tabs or menus. Press a keyboard shortcut, type what you're looking for, and jump to that feature instantly.

Think of it like a spotlight search for your tools: looking for the SOQL Builder? Type "soql" and hit Enter. Want to jump to the Records tab? Type "records". Need to look up a specific record ID? Paste the ID and the palette recognizes it automatically. Everything is keyboard-driven and stateless — no history, no retained state between uses — so you get consistent, predictable behavior every time.

How to Open

The Command Palette is accessible from anywhere in TrackForcePro. You have two ways to open it:

  1. Keyboard Shortcut (Fastest): Press Ctrl+K (Windows/Linux) or Cmd+K (Mac) inside the extension workspace popup. The palette appears instantly with a search field focused and ready for input. On Salesforce pages, the Command Palette shortcut must be assigned manually in Settings → General → Keyboard Shortcuts (not assigned by default).
  2. Click the Trigger Button: Look for the Command Palette button on Salesforce pages (the position is configurable in Settings → Sidebar). Click it to open the palette.

Once open, the palette shows a search box at the top with a list of available commands below. Start typing to filter the list. Your query matches command names, descriptions, and keywords.

Press Escape to close the palette immediately without executing anything. The palette closes automatically after you execute a command.

Built-In Commands

TrackForcePro includes 13+ built-in commands covering all major tabs and features:

Command Action
Switch to Audit Trail Opens the Audit Trails tab. Browse and filter org-level audit trail events.
Switch to SOQL Builder Switches to the SOQL Builder tab. Write, test, and execute SOQL queries with syntax highlighting and suggestions.
Switch to GraphQL Builder Opens the GraphQL Builder tab. Compose and execute GraphQL queries against Salesforce with schema introspection.

Switch to Org Tools Switches to the Org Tools tab. Access org metadata, limits, custom settings, and other utilities.
Switch to Platform Events Opens the Platform Events tab. Monitor and test Platform Event publishing and subscriptions.

Open Show All Data Opens the Show All Data overlay on the current Salesforce record page. Shows all fields, values, types, and editability.
Open LWC Explorer Launches the LWC Explorer component and network trace inspector on the current Salesforce page.
Toggle Sidebar Toggles the visibility of the TrackForcePro sidebar on Salesforce pages.
Copy Record ID Copies the current record's 18-character ID to your clipboard. No modal, instant copy.
Open Setup Search Opens Salesforce Setup with the search field pre-focused, so you can immediately search for pages, objects, and settings.

Type the name or part of the name of any command to filter the list. For example, typing "soql" shows the SOQL Builder command. Typing "switch" shows all tab-switching commands.

Record ID Detection

One of the most powerful Command Palette features is automatic record ID detection. You don't need to type a command — just paste or type a Salesforce record ID directly.

The palette recognizes both 15-character and 18-character record IDs. For example:

  • 15-char: 001D000000IRFmaIAH
  • 18-char: 001D000000IRFmaIAH000 (includes checksum)

When you paste or type an ID, the palette immediately detects it as a record lookup and shows "Look Up Record" as an option. Press Enter to execute the lookup. The palette switches to the Records tab and loads the record details instantly.

This is perfect for workflows where you need to inspect a specific record: paste the ID, press Enter, and you're in the Records tab viewing that record. No tab-switching, no manual searches.

Time-Saving Tip: Copy a record ID from any Salesforce page (right-click → "Copy Record ID"), switch to TrackForcePro, press Ctrl+K, paste the ID, and press Enter. You're inspecting the record in TrackForcePro within seconds.

Keyboard Navigation

The Command Palette is designed for 100% keyboard operation. Your hands never leave the keyboard.

Key Action
Ctrl+K / Cmd+K Open the Command Palette. Works from any tab or Salesforce page.
↑ / ↓ Arrow Keys Navigate up or down through the command list. The selected command is highlighted.
Enter Execute the currently selected command or record lookup. Closes the palette and performs the action.
Escape Close the palette immediately without executing anything. Returns focus to the previous context.
Type to Search As you type, the command list filters in real-time. Results update with each keystroke. Match is case-insensitive.

The palette remembers nothing between sessions. Every time you open it, you start with a blank search and the full command list. This stateless design prevents confusion and ensures consistent behavior.

Tips & Tricks

💡 Tip 1: Learn the Keyboard Shortcut
Ctrl+K is blazingly fast once it becomes muscle memory. If you use TrackForcePro multiple times per day, learning this shortcut will save you significant time over a week.
💡 Tip 2: Tab-Switching Speed Champion
The Command Palette is the fastest way to switch tabs. Ctrl+K, type the first letter of the tab name, press Enter — done. Faster than clicking tabs.
💡 Tip 3: Record ID Lookup Workflow
Copy a record ID from any Salesforce page using "Copy Record ID" from the context menu. Ctrl+K, paste, Enter. You're looking at the record in 2 seconds.
💡 Tip 4: Partial Matching Works
Don't need to type the full command name. Type "rec" to find "Records", "rest" to find "REST Explorer", or "aud" to find "Audit Trail". Substring matching is your friend.
💡 Tip 5: No History to Clutter Your Brain
The palette doesn't store search history or previous commands. This simplicity is intentional — you always get the same clean list and no accumulated clutter from old searches.
💡 Tip 6: Customize the Shortcut in Settings
If Ctrl+K conflicts with your browser or OS shortcuts, you can rebind the Command Palette shortcut in Settings → General → Keyboard Shortcuts. Pick a combination that works for your setup.
💡 Tip 7: Position Independence
The Command Palette trigger button position (Right/Left/Bottom) is independent from the sidebar and Show All Data button. Configure all three separately in Settings → Sidebar for your perfect layout.
