🏢 Org Tools Guide Workspace Tab

Comprehensive org-level administration tools for custom labels, org limits, metadata exploration, debug logs, and deployments.

What It Does

Org Tools is a comprehensive suite of organization-level administration utilities. Manage custom labels and translations, monitor org limits and governor quotas, search metadata across your org, manage debug logs, and deploy metadata between orgs. Promoted to a top-level tab in v1.8.0, Org Tools provides seven specialized panels accessible from a toggle bar.

How to Access

Open the TrackForcePro workspace popup and click the Org Tools tab (🏢 icon) in the main tab bar. The tab displays a panel toggle bar at the top with buttons for each of the seven tool panels. Click any button to switch between panels.

You can also access org tools from the Sidebar or Command Palette on any Salesforce page.

Custom Labels Panel

View, search, export, and import custom labels directly from TrackForcePro without visiting Setup.

Key Features

  • Search & Filter — Find labels by name or category
  • 📤 Export — Download labels as CSV for bulk editing or translation
  • 📥 Import — Upload translated labels back to your org
  • 🌐 Translate — Mark labels for translation workflows
  • Bulk Operations — Update multiple labels at once
Translation Workflow: Export labels → translate externally or with your team → import back to your org. This is the safest way to manage translations at scale without manual Setup entry.

Org Limits Panel

Real-time dashboard of your Salesforce org's governor limits and usage. Displays current utilization as a percentage of each limit.

Monitored Limits

  • API Calls — Daily limit on REST/Soap/Tooling API calls
  • File Storage — Files and documents quota
  • Data Storage — Records and custom objects quota
  • Metadata API — Concurrent deploy/retrieve limits
  • Batch Size — Records per batch in Batch Apex
  • Soql Row Count — Query result set limit

Data updates in real time from the Salesforce Limits REST endpoint. Use this to track API usage and plan bulk operations.

Metadata Explorer Panel

Broad metadata search across your entire org. Find custom objects, Apex classes, flows, permission sets, custom fields, and more by API name.

What Makes It Different from Setup Search

Feature Metadata Explorer Setup Search
Search Scope Multiple metadata types (objects, classes, flows, etc.) Salesforce pages and links
Query By API name or partial substring Page title or label
Results Detail Full metadata details and attributes Quick page navigation
Use Case Deep dives and bulk research Quick Setup page location
Tip: Use Metadata Explorer when you need comprehensive cross-object searches. Use Setup Search (native Salesforce, accessed via sidebar in TrackForcePro) for quick "where is this page?" lookups.

Debug Logs Panel

Manage debug log trace flags without visiting Setup. Create new logs for specific users, view active logging levels, and download log content.

Key Features

  • View Active Trace Flags — See which users have debug logging enabled and at what level
  • Create New Trace Flags — Enable logging for a user without going to Setup
  • Set Log Levels — Choose Debug, Info, Warn, or Error level per user
  • Download Logs — Download log files to review locally
  • View Log Content — Inline log viewer with syntax highlighting
Debug Log Limits: Salesforce stores the last 20 debug logs per org (or 1000 lines per log, whichever is smaller). Old logs are automatically deleted as new ones are created.

Deployment Tools Panel

Deploy metadata between Salesforce orgs, validate deployments before committing, and track deployment history.

Common Workflows

  • Sandbox to Production — Package changes from dev sandbox and deploy to production
  • Validation Deploy — Test deployment without committing. Useful for risky changes.
  • Rollback — Restore previous deployment state by re-deploying old versions
  • Metadata Subset — Deploy only specific components (classes, flows, objects) instead of everything
🚀 Deploy Button: Click the Deploy button to start a new deployment. Select your source org, target org, and components. Confirm the deployment, and TrackForcePro handles the rest via the Metadata API.

Tips & Best Practices

  • Use Metadata Explorer for broad org research — Finding all references to a custom field or flow? Metadata Explorer is faster than multiple Setup searches.
  • Export custom labels for team translation — Rather than translating in Setup, export to CSV, share with your team, and import back. Avoids concurrent-edit conflicts.
  • Monitor org limits before bulk operations — Check the Org Limits panel before running large data loads or Batch Apex jobs. Prevents hitting limits mid-job.
  • Create debug logs from TrackForcePro — Faster than Setup when you need to debug a specific user's action. No Setup navigation needed.
  • Validate deployments before production — Use the Validation Deploy option to test metadata changes without committing. Catches errors before they go live.
  • Setup Search is still your friend — For quick "where is X page?" lookups, Setup Search (native Salesforce, accessed via sidebar) is faster than Metadata Explorer. Use the right tool for the right job.
