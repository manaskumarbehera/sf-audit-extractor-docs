  • ⚡ Show All Data — floating inspector button injected directly on Salesforce record pages
  • 🔍 Command Palette (Ctrl+K / Cmd+K) — global search with 13+ commands and record ID auto-detection
  • 🖱️ Context Menu — right-click on SF pages for Show All Data, Field History, Sharing, SOQL Builder, Copy Record ID
  • ✏️ Inline Field Editing — double-click any field in All Fields inspector to edit with type-aware inputs
  • 🔗 Record Breadcrumbs & Pinned Records — navigate lookup chains with breadcrumb trail, pin favorites
