Key Features

🔍 Audit Trails

Fetch, filter by category, search, and export Setup Audit Trail logs to CSV for compliance and change tracking

📊 SOQL Builder

Visual query builder with searchable objects, field suggestions, guidance engine, multi-tab editing, and 5 export formats

🔗 GraphQL Builder

Three-screen workflow with CodeMirror 6 editor, autocomplete, draggable splitters, pagination, and export

🔬 Records

Look up records by ID, view all fields, edit inline, navigate lookups with breadcrumbs, pin favourites, bulk scan 200 IDs

👥 User Manager

Search users, manage permissions, freeze/unfreeze accounts, compare users side-by-side, and bulk update

🔐 Security Manager

Browse Profiles, Permission Sets, and Permission Set Groups with search and assignment drill-down

🏢 Org Tools (7 Panels)

Login As, Custom Labels (with bulk import/export & translations), Deployments, Scheduled Jobs, Debug Logs, Org Limits, Admin Utils

🎨 Org & Favicon

Design custom favicons per org with 8 shapes, colours, and badge labels to distinguish environments at a glance

📡 Platform Events

Real-time CometD streaming for Platform Events with filtering, pause/resume, and auto-scroll

💬 Lightning Message Service

Subscribe to and publish LMS channel messages with JSON payload editor and log filtering

👁️ Show All Data

Floating inspector on every record page: see all fields (including hidden), search, edit inline, navigate lookups

🧭 TF Sidebar & Command Palettes

Floating sidebar for page shortcuts, a launcher/workspace Command Palette for cross-feature navigation, and a separate Command Palette (Page) for page-aware Salesforce actions. Use Setup Search when you want setup navigation plus metadata search.

🗂️ Setup Search & Metadata Search

Search Setup and metadata by API name from Salesforce pages, then hand off into the workspace for deeper inspection

What's New

v1.9.7 — March 25, 2026 (Latest)

  • Virtualized SOQL result table — new SoqlVirtualTable engine renders only visible rows for smooth scrolling through 1000+ rows and 50+ columns. Three density modes (Normal, Compact, Dense), column pinning, per-column filtering, global search, Smart View, and full-screen/focus modes.
  • Compact result toolbar — 30px icon-based toolbar replaces the old text-heavy action bar with density selector, export dropdown (JSON/CSV/TSV), smart view toggle, and pop-out button.
  • Pop-out results window — open SOQL results in a dedicated browser tab with full toolbar and virtual table support.
  • Collapsible query editor — toggle button collapses the SOQL editor to give more vertical space to result tables.
  • 6 suggestion bug fixes — stale dropdown after execution, wrong-context insertion, unreachable subquery FROM, async race condition, and more.

v1.9.6 — March 25, 2026

  • LMS Explorer redesigned — channels now appear in the tree pane and selecting one shows its schema, publish form, and activity feed in the detail pane, matching how components work.
  • LMS Beta phase — a purple "Beta" badge on the LMS pill and an informational banner signals that LMS support is still being refined.
  • Stability fix — resolved "Extension context invalidated" console errors after extension reload.

v1.9.5 — March 25, 2026

  • LMS bridge rewrite — publish now works on live Salesforce pages via real Aura $A framework integration with four priority strategies.
  • LWC Explorer live mode — 3-second trace polling, labeled Scan/Live buttons, clear traces, and persistent dock mode.
  • SOQL relationship links — dot-notation fields like Account.Name render as clickable links with Copy Value action.
  • Two-row header — tab bar now scrolls horizontally, preventing label overlap on narrow viewports.

v1.9.4 — March 24, 2026

  • Smarter LWC Explorer traces — page traffic is now separated into real imperative Apex, UI API, GraphQL, and Fetch/API so non-Apex framework noise no longer shows up as Apex.
  • Large in-page code editor — both Edit Code and Edit Apex now open the full-sized editor window directly from the explorer.
  • Branded troubleshooting surfaces — the LWC Explorer, command palettes, and workspace/launcher surfaces now use the TrackForcePro icon and a more consistent header treatment.

v1.9.3 — March 24, 2026

  • Release sync patch — version bump, docs sync, and store-submission support updates.
  • SOQL Builder robustness — smaller hardening fixes and supporting test updates.

v1.8.8 — March 21, 2026

  • Settings architecture refactor — the first shared settings schema/service is now in place, starting with Sidebar panel settings for a more consistent Save & Apply model.
  • Metadata search improvements — Setup Search and Metadata Explorer now support broader API-name discovery, especially for flows and security-related metadata.
  • New Training Guide — a role-based learning path for admins, developers, support teams, and trainers is now part of the docs hub.

v1.8.6 — March 20, 2026

  • Independent Button Positioning — Show All Data button and Command Palette trigger now have their own position settings (Right / Left / Bottom), independent of the sidebar.
  • Command Palette Fix — Ctrl+K now works more reliably, with a popup-safe alternative and visual trigger.
  • Custom Labels Fix — edit, import, and translation deployment now use corrected Salesforce API flows.

v1.7.x — March 2026

  • Settings Export & Import — Backup and restore all settings, query history, and per-org data as JSON
  • Searchable Object Selector — SOQL Builder combobox with substring filtering and keyboard navigation
  • Show All Data Improvements — Better performance, breadcrumb navigation, and improved field display
