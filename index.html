Key Features

🔍 Audit Trails

Fetch, filter by category, search, and export Setup Audit Trail logs to CSV for compliance and change tracking

📊 SOQL Builder

Visual query builder with searchable objects, field suggestions, guidance engine, multi-tab editing, and 5 export formats

🔗 GraphQL Builder

Three-screen workflow with CodeMirror 6 editor, autocomplete, draggable splitters, pagination, and export

🔬 Records

Look up records by ID, view all fields, edit inline, navigate lookups with breadcrumbs, pin favourites, bulk scan 200 IDs

👥 User Manager

Search users, manage permissions, freeze/unfreeze accounts, compare users side-by-side, and bulk update

🔐 Security Manager

Browse Profiles, Permission Sets, and Permission Set Groups with search and assignment drill-down

🏢 Org Tools (7 Panels)

Login As, Custom Labels (with bulk import/export & translations), Deployments, Scheduled Jobs, Debug Logs, Org Limits, Admin Utils

🎨 Org & Favicon

Design custom favicons per org with 8 shapes, colours, and badge labels to distinguish environments at a glance

📡 Platform Events

Real-time CometD streaming for Platform Events with filtering, pause/resume, and auto-scroll

💬 Lightning Message Service

Subscribe to and publish LMS channel messages with JSON payload editor and log filtering

👁️ Show All Data

Floating inspector on every record page: see all fields (including hidden), search, edit inline, navigate lookups

🧭 TF Sidebar & Command Palette

Floating sidebar with shortcuts on every Salesforce page. Ctrl+K command palette with 13+ commands and record ID auto-detection

🗂️ Setup Search & Metadata Search

Search Setup and metadata by API name from Salesforce pages, then hand off into the workspace for deeper inspection

What's New

v1.8.8 — March 21, 2026 (Latest)

  • Settings architecture refactor — the first shared settings schema/service is now in place, starting with Sidebar panel settings for a more consistent Save & Apply model.
  • Metadata search improvements — Setup Search and Metadata Explorer now support broader API-name discovery, especially for flows and security-related metadata.
  • New Training Guide — a role-based learning path for admins, developers, support teams, and trainers is now part of the docs hub.

v1.8.6 — March 20, 2026

  • Independent Button Positioning — Show All Data button and Command Palette trigger now have their own position settings (Right / Left / Bottom), independent of the sidebar.
  • Command Palette Fix — Ctrl+K now works more reliably, with a popup-safe alternative and visual trigger.
  • Custom Labels Fix — edit, import, and translation deployment now use corrected Salesforce API flows.

v1.7.x — March 2026

  • Settings Export & Import — Backup and restore all settings, query history, and per-org data as JSON
  • Searchable Object Selector — SOQL Builder combobox with substring filtering and keyboard navigation
  • Show All Data Improvements — Better performance, breadcrumb navigation, and improved field display
