Three Work Surfaces

🚀 Launcher
Quick-access popup for instant queries, record lookups, and feature shortcuts.
  • Search records by ID
  • Quick feature toggles
  • Org context detection
💻 Workspace Tab
Full-featured development environment organized by mode: Explore, Debug, Analyze.
  • 9 tabs with sub-panels
  • Query builders & explorers
  • Settings & preferences

Context-aware tools that appear directly on Salesforce pages you're working with.
  • Sidebar quick tools
  • Show All Data panel

🔍 Explore — Query & Discover

📝

SOQL Builder

Visual query builder with searchable objects, field suggestions, autocomplete, inline filters, and intelligent guidance engine. Execute, export in 5 formats, pin queries.

🔗

GraphQL Builder

Three-screen workflow with CodeMirror 6 syntax highlighting, autocomplete, live validation, pagination, and formatted export.

⚡

REST Explorer

Send any Salesforce REST API request, organize collections, chain requests, import cURL commands, and browse the API catalog.

🔬

Records Lookup

Search by 15 or 18-character ID, view all fields, edit inline with type-aware inputs, navigate lookups with breadcrumbs, pin favorites, bulk scan up to 200 IDs.

👥

User Manager

Search users, manage permissions, freeze/unfreeze accounts, compare side-by-side, bulk update, and drill into permission set assignments.

🔐

Security Manager

Browse profiles, permission sets, and permission set groups. Search, filter, and drill down into assignments and dependencies.

🐛 Debug — Monitor & Troubleshoot

📡

Platform Events

Real-time CometD streaming for Platform Events. Filter by channel, pause/resume live flow, auto-scroll, and inspect event payloads in real time.

💬

Lightning Message Service

Subscribe to and publish LMS channel messages. View real-time activity, edit JSON payloads, filter logs by channel and message type.

🔍

LWC Explorer

In-page network inspector for Lightning Web Components. Trace real Apex, UI API, GraphQL, and Fetch calls. Live mode with 3-second polling, filtering by type, and request/response inspection.

📊 Analyze — Audit & Monitor

🔍

Audit Trails

Fetch Setup Audit Trail logs, filter by 10+ categories, search by user/date/action, and export to CSV for compliance and change tracking.

💚

Org Health

Monitor org limits, storage usage, API calls, concurrent requests, and batch jobs. Visual indicators show health status at a glance.

🏢

Org Tools (7 Panels)

Login As, Custom Labels (with bulk import/export and translations), Deployments, Scheduled Jobs, Debug Logs, Org Limits, and Admin Utilities.

🎨

Org & Favicon

Design custom favicons per org with 8 shapes, colors, and custom badge labels. Instantly distinguish production, sandbox, and dev environments in your browser tabs.

✨ In-Page Tools — Right Where You Need Them

👁️

Show All Data

Floating panel on every record detail page. See all fields including hidden ones, search and filter, edit inline, and navigate related records via lookups.

🧭

Sidebar & Shortcuts

Floating sidebar on Salesforce pages with quick-access shortcuts for common admin tasks. Toggle with custom keyboard shortcut, customize entries in Settings.

⌨️

Command Palette

Global command palette (Ctrl+K / Cmd+K) for cross-feature navigation. Type to search 13+ commands, discover features, jump between tabs instantly.
