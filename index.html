v1.6.0 — March 16, 2026

  • UI Modernization — glassmorphism, refined typography (Inter), smoother animations, modern color palette
  • Modern Tab Design — pill-shaped tabs with active glow, hover micro-interactions

  • Docs Redesign — all documentation pages modernized with gradient hero, glassmorphism cards

  • Context Menu — right-click on SF pages for Show All Data, Field History, Sharing, SOQL Builder, Copy Record ID

  • Record Breadcrumbs & Pinned Records — navigate lookup chains with breadcrumb trail, pin favorites
