Things other Salesforce extensions can't do

TrackForcePro reads what's actually happening on the page — and turns it into answers. Two flagship capabilities lead the v2.10.0 release.

In LWC Explorer 🩺

Live-Page Diagnostics

The LWC Explorer captures real Apex, UI API, GraphQL, and Fetch traffic on any Lightning page — then makes it make sense. Understand and fix a page without opening a single dev console.

  • ⏱️ Performance Profiler
  • 🔁 Capture → Replay
  • 🧩 Errors → Root Cause
  • 🔬 Explain this Page (X-Ray)
  • 🌊 Flow & Automation Trace
🗂️

Data Tooling at Scale

Move and audit records without leaving the org: CSV import for any SObject (Insert/Update/Upsert/Delete) with mapping, preview & error reports — plus bulk mutate/delete straight from the SOQL results grid. Production-gated.

💾

Saved Query Library

Save, search, and reload SOQL & GraphQL queries. Export/import the whole library as JSON.

📊

Field Fill-Rate Analyzer

Find unused fields on any object via batched COUNT() aggregates — plan cleanups with evidence.

⚖️

Permission-Set & Profile Compare

Diff CRUD and field-level security across permission sets and profiles side by side. Spot access drift instantly before it becomes an audit finding.

🔍 Explore — Query & Discover

📝

SOQL Builder

Visual query builder with searchable objects, field autocomplete, inline filters, a guidance engine, a virtualized 1000-row grid, and export in 5 formats.

🔗

GraphQL Builder

CodeMirror 6 editor with syntax highlighting, autocomplete, live validation, pagination, and formatted export.

⚡

REST Explorer

Send any Salesforce REST request, organize collections, chain calls, import cURL, and browse the API catalog.

🗺️

Schema Explorer

Browse objects, fields, data types, and relationships. Inspect metadata and jump to Setup with reliable encoded deep links.

🔬

Records Lookup

Search by 15/18-char ID, view all fields including hidden ones, edit inline, navigate lookups with breadcrumbs, pin favorites, bulk-scan up to 200 IDs.

👥

Users & Security

Manage users, freeze/unfreeze, compare side by side, and browse profiles, permission sets, and permission-set groups with their assignments.

🐛 Debug — Monitor & Troubleshoot

📡

Platform Events

Real-time CometD streaming. Filter by channel, pause/resume the live flow, auto-scroll, and inspect event payloads as they arrive.

💬

Lightning Message Service

Subscribe to and publish LMS channel messages, edit JSON payloads, and watch real-time activity — right inside the LWC Explorer overlay.

🔍

LWC Explorer

In-page network inspector for Lightning Web Components. Trace real Apex, UI API, GraphQL, and Fetch calls with a 3-second live mode and type filtering.

📊 Analyze — Audit & Monitor

🛡️

Audit Trail

Fetch Setup Audit Trail logs, filter by 10+ categories, search by user/date/action, and export to CSV for compliance and change tracking.

💚

Org Health

Monitor org limits, storage, API calls, concurrent requests, and batch jobs with at-a-glance health indicators.

🏢

Org Tools

Seven panels: Login As, Custom Labels (bulk import/export & translations), Deployments, Scheduled Jobs, Debug Logs, Org Limits, and Admin Utilities.

🎨

Org & Favicon

Design a custom favicon per org with shapes, colors, and badge labels — instantly tell production, sandbox, and dev tabs apart.
