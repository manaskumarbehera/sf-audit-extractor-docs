Overview

TrackForcePro is a Salesforce toolkit focused on quick access, safe multi-org workflows, and productivity for admins/developers.

Core principle: data stays local to your browser unless you explicitly export content.

Installation

  1. Install from Chrome Web Store.
  2. Open any Salesforce page.
  3. Use the floating TF sidebar button to launch tools.

Quick Start

  • Open SOQL Builder to run your first query.
  • Use Security Manager to inspect profile/permission grants.
  • Set org-specific favicon to avoid wrong-org edits.
