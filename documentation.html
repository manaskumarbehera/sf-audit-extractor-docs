📋 Audit Trail Extraction

Fetch and analyze Salesforce audit logs. View who changed what, when, and export to CSV for compliance and troubleshooting.

Open the Audit Trails Tab

Click the Audit Trails icon in the TrackForcePro popup to view the audit log interface.

Set Date Range

Select a start and end date. Salesforce typically retains audit logs for 6 months to 10 years depending on your edition.

Filter Results

Filter by user, object type, action (Create, Update, Delete, etc.), or search by field name to narrow results.

Fetch Audit Logs

Click the fetch button to retrieve logs from your Salesforce org. Results appear in a searchable table.

Export to CSV

Click Export to download the results as a CSV file for use in Excel, Google Sheets, or other analysis tools.

💡 Pro Tip: Use the search box to find specific audit records. Perfect for tracking field changes on specific records or monitoring user activity.

🔍 SOQL Builder

Write and test SOQL queries without leaving Salesforce. Visual builder helps you construct queries with autocomplete guidance.

Click SOQL Builder Tab

Open the SOQL Builder from the main popup menu.

Select an Object

Choose a Salesforce object (Account, Contact, Lead, etc.) from the searchable dropdown. Type to filter the list.

Add Fields

Click the plus icon to add fields from the object. Select common fields or search for specific ones. Click checkboxes to include in your query.

Build Filters (Optional)

Add WHERE conditions to limit results. Set field, operator (equals, contains, greater than, etc.), and value.

Execute Query

Click Execute to run the SOQL query and see results instantly in a table below.

Export or Edit Results

Export results to CSV, or double-click any cell to edit inline directly from the results table.

💡 Pro Tip: Copy your generated SOQL query to use in the Salesforce Developer Console or Reports. The visual builder shows syntax guidance as you build.

⚡ GraphQL Builder

Build and test GraphQL queries for Salesforce using a powerful code editor with autocomplete and formatting.

Open GraphQL Builder

Click the GraphQL Builder tab in the popup.

Write Your Query

Type or paste your GraphQL query in the editor. Get real-time syntax hints as you type.

Format Query

Click the Format button to auto-format your query for readability.

Execute

Click Execute to run the query. Results display in a formatted JSON response below.

Export Results

Export the GraphQL response to JSON or copy it to your clipboard.

💡 Pro Tip: Use GraphQL for complex nested queries. The autocomplete feature learns from your org's schema to provide intelligent suggestions.

📊 Data Explorer

Look up any Salesforce record by ID, view all fields, edit inline, navigate between related records, and compare records side-by-side.

Open Data Explorer

Click the Data Explorer tab in the popup.

Search for a Record

Enter a Salesforce record ID (15 or 18 character) in the search box and press Enter. Or use the single/bulk record scanner for more control.

View All Fields

See every field on the record—including read-only fields not visible in Salesforce. Results are paginated (100 fields per page).

Edit Fields Inline (Optional)

Double-click any editable field to change its value. Press Enter to save or Escape to cancel. Changes update Salesforce immediately.

Navigate Related Records

Click a lookup field to jump to the related record. A breadcrumb trail keeps track of your navigation history.

Pin Frequently-Used Records

Click the pin icon on any record to save it. Pinned records appear at the top for quick re-access.

💡 Pro Tip: Use Bulk Record Scanner to search up to 200 record IDs at once. Results show in a sortable table. Compare records side-by-side to spot differences.
⚠️ Inline Edit Security: In Settings > General, you can disable inline editing in production orgs to prevent accidental changes. This setting protects production data while allowing edits in sandboxes.

🔐 Security Manager

Browse and analyze Salesforce security configurations: Profiles, Permission Sets, and Permission Set Groups.

Open Org Tools Tab

Click Org Tools in the main popup, then select Security Manager.

Browse Profiles

View all Salesforce profiles in your org. See which objects and fields each profile can access.

View Permission Sets

Check individual permission sets and understand what permissions they grant users.

Inspect Permission Set Groups

See groups and the permission sets they contain.

💡 Pro Tip: Use this tool to audit user permissions and understand your org's security model without manual Salesforce UI navigation.

👥 User Manager

Search users, view detailed profiles, freeze/unfreeze accounts, and see permission assignments at a glance.

Open Org Tools > User Manager

Navigate to Org Tools and select User Manager.

Search Users

Type a name or email in the search box to find users. Filter by active/inactive status.

View User Details

Click a user to see full profile info: email, role, profile, permission sets, and status.

Freeze or Unfreeze Users

Click the freeze/unfreeze button to temporarily disable a user account without deleting it.

Check Permission Matrix

View which permission sets and profiles are assigned to each user.

💡 Pro Tip: Use User Manager to quickly deactivate contractors or temp users without accessing Salesforce Setup.

🛠️ Org Tools

Quick access to org utilities: Login As, Custom Labels, Deployments, Debug Logs, Org Limits, and Scheduled Jobs.

Open Org Tools

Click the Org Tools tab in the popup.

Login As Another User

Search for a user and click Login As to assume their identity for troubleshooting.

Browse Custom Labels

View all custom labels in your org. Search by label name or category.

Monitor Deployments

See recent deployment history with status and details.

Check Debug Logs

View recent debug logs and download them for analysis.

Monitor Org Limits

See API calls used, data storage, file storage, and other org limits in real-time.

View Scheduled Jobs

See all scheduled batch jobs, their status, and next execution time.

💡 Pro Tip: Check Org Limits regularly to monitor API usage and storage. Use Debug Logs to troubleshoot code execution issues.

📡 Platform Events

Subscribe to and publish Salesforce Platform Events in real-time. Monitor event streams for debugging and testing.

Open Platform Events Tab

Click Platform Events in the popup menu.

Select an Event

Choose a Platform Event from your org's list.

Subscribe to Events

Click Subscribe to listen to incoming events. Events appear in real-time as they're published.

Publish a Test Event

Enter event payload data and click Publish to send a test event to your subscribers.

Monitor Event Stream

View all events received in JSON format. Clear logs or download for analysis.

💡 Pro Tip: Use Platform Events for testing integrations and debugging event-driven workflows without manual triggering.

💬 Lightning Message Service

Subscribe to and publish LMS channels for testing component communication in Lightning.

Open LMS Tab

Click Lightning Message Service (LMS) in the popup.

Select a Channel

Choose an LMS channel from your org.

Subscribe to Channel

Click Subscribe to listen for messages on the selected channel.

Publish Messages

Enter message payload and click Publish. Subscribers receive the message instantly.

View Message Log

All sent and received messages appear in a scrollable log with timestamps.

💡 Pro Tip: Use LMS testing to verify Lightning component communication works correctly without manually triggering actions.

👁️ Show All Data

Floating button appears on every Salesforce record. Click to see all fields—including hidden ones—and edit them directly on the page.

Navigate to Any Salesforce Record

Open any record page in Salesforce (Account, Contact, Opportunity, etc.).

Click the Show All Data Button

A floating TrackForcePro button appears in the top corner. Click it to open the All Fields inspector.

Search Fields (Optional)

Use the search box to find specific fields by name or value.

Edit Fields

Double-click any field value to edit. Type your changes and press Enter to save instantly.

Navigate Related Records

Click lookup fields to jump to related records. Breadcrumbs track your navigation history. Click breadcrumbs to go back.

💡 Pro Tip: Use Show All Data to see formula results, system fields (CreatedDate, CreatedById), and fields hidden by page layouts.

⚙️ TrackForcePro Sidebar

Quick shortcuts appear as a sidebar on every Salesforce page. Customize shortcuts to your favorite tools in Settings.

Sidebar Appears Automatically

A floating sidebar with shortcuts is visible on all Salesforce pages.

Click Shortcuts

Click any shortcut to trigger the action (Show All Data, Field History, SOQL Builder, etc.).

Customize Shortcuts

Open Settings > Sidebar and configure which shortcuts appear on your sidebar.

Hide Sidebar (Optional)

Click the minimize icon to hide the sidebar. It reappears when you navigate to a new Salesforce page.

💡 Pro Tip: Customize shortcuts in Settings to match your workflow. Add frequently-used tools to the sidebar for quick access.

⌨️ Command Palette

Global command search accessible from any Salesforce page. Press Ctrl+K (or Cmd+K on Mac) to open and search for commands.

Press Ctrl+K (or Cmd+K)

From any Salesforce page, press Ctrl+K to open the Command Palette.

Search Commands

Type to search: "Audit," "SOQL," "Show All Data," "Security," etc. Matching commands appear instantly.

Navigate Results

Use arrow keys to move between results. Press Enter to execute the selected command.

Record ID Lookup

Type a record ID (15-18 characters) to auto-trigger a record lookup in Data Explorer.

Close Palette

Press Escape to close the Command Palette anytime.

💡 Pro Tip: The Command Palette supports 13+ built-in commands. Customize shortcut keys in Settings > General > Configurable Keyboard Shortcuts.

🖱️ Context Menu

Right-click on any Salesforce record page for quick access to Show All Data, Field History, Sharing, SOQL Builder, and more.

Right-Click on a Salesforce Page

Navigate to any Salesforce record and right-click the page.

Use Context Menu Options

A TrackForcePro menu appears with 5 options:

Available Actions

Show All Data - Open the field inspector. Field History - View field change history. Sharing - Check record sharing settings. SOQL Builder - Write queries for the current object. Copy Record ID - Copy the record ID to clipboard.

💡 Pro Tip: Right-click is faster than navigating the popup menu for quick actions on record pages.

⚙️ Settings

Customize TrackForcePro behavior, appearance, and keyboard shortcuts. Settings are saved per browser profile.

General Settings: Disable inline edit in production orgs, configure keyboard shortcuts (Ctrl+K toggle, Show All Data, etc.), set theme preference.
Org & Favicon: Manage org logos and custom favicons. Show org icons on browser tabs for easy identification.
Appearance: Choose light, dark, or auto theme. Customize colors and font sizes for accessibility.
Sidebar: Configure which shortcuts appear on the Salesforce sidebar. Reorder or disable individual shortcuts.
Layout: Adjust column widths in tables. Settings persist across sessions.
Backup: Export or import your entire TrackForcePro configuration. Useful when switching browsers or computers.
💡 New in v1.8.0: Disable Inline Edit in Production—blocks inline edits on production orgs while allowing them in sandboxes. Configurable Keyboard Shortcuts—set custom key combos for Toggle Sidebar, Show All Data, and Command Palette.
