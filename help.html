💬 Frequently Asked Questions

How do I install TrackForcePro?
Install from the Chrome Web Store or from Firefox Add-ons. Edge remains a docs placeholder for now. See our Quick Start Guide for detailed steps.
Is my Salesforce data secure?
Yes — TrackForcePro processes all data locally in your browser. No data is sent to external servers. The extension communicates only with your Salesforce org using your existing session credentials. See our Privacy Policy for full details.
Which Salesforce editions are supported?
TrackForcePro works with all Salesforce editions: Developer, Professional, Enterprise, and Unlimited. Both Production and Sandbox environments are fully supported.
How do I use inline field editing?
Double-click any editable field in the Records tab, SOQL results table, or Show All Data panel. A type-aware editor appears (text input, dropdown for booleans, textarea for long text). Press Enter to save or Esc to cancel. Changes write directly to Salesforce via the REST API.
How do I protect production orgs from accidental edits?
Go to Settings > Org Management and toggle the inline editing safety switch for each org. When disabled for an org, all inline edit entry points (Records, SOQL Builder, and Show All Data) are blocked and a warning notification appears. Sandboxes remain fully editable.
Can I customise my keyboard shortcuts?
Yes — open Settings > General. You'll see click-to-record shortcut groups for main actions, page actions, workspace tabs, Records tools, and Org Tools panels. Click any button, press your desired key combination, and it's saved automatically. Reset individual shortcuts to defaults with the reset button.
Can I move the floating buttons to a different position?
Yes — open Settings > Sidebar. Each floating element has its own position setting: the Sidebar (Left/Right), Show All Data button (Left/Right/Bottom), and Command Palette trigger (Left/Right/Bottom). Bottom places buttons as horizontal tabs along the bottom edge of the screen. Changes apply immediately on all open Salesforce pages.
How do I distinguish between orgs in my browser tabs?
Use the Org & Favicon tab to design custom favicons per org. Choose distinct colours, shapes, and badge labels (like "PRD", "DEV", "UAT"). Click Save & Apply — all open Salesforce tabs for that org update instantly with the new favicon, tab title, and logo colour.
How do I export audit logs?
Navigate to the Audit Trails tab, use the category filter and search to narrow results, then click the Export button to download as CSV.
How do I manage custom labels in bulk?
In Org Tools > Custom Labels, use the bulk import feature to upload labels from CSV or JSON. Bulk export downloads all labels. The full translation workflow lets you export a template, add translations, and deploy — all from one panel.
How do I back up my settings?
Go to Settings > Backup and click Quick Export. A JSON file downloads containing all your settings, query history, shortcuts, appearance, and per-org data. To restore, click Quick Import and select the JSON file.
The extension shows "Not Connected" — what do I do?
Make sure you have an active, logged-in Salesforce tab open. The extension detects your session from open tabs. Try: refresh the Salesforce page, log out and back in, then reopen the extension popup. If the service worker has crashed, the extension automatically recovers using cookie fallback.

🔧 Troubleshooting

IssueSolution
Extension not appearing on Salesforce Ensure you're on a Salesforce page (*.salesforce.com, *.force.com, or *.lightning.force.com). The TFP button appears at the bottom-right on record pages. The sidebar can be toggled with Ctrl+Shift+S.
Session expired or "No access token" Log out and log back into Salesforce. Refresh the page. The extension detects the new session automatically. If the service worker crashed, the extension recovers using cookie fallback — you may see a brief "degraded mode" toast.
Queries not executing or returning errors Verify you have API access enabled in your Salesforce profile. Check object and field names for accuracy. Review the error message. Use the SOQL Guidance Engine panel for real-time query validation hints.
Objects or fields not loading in builders Check your internet connection. Verify you're logged into Salesforce. Click the Refresh button. Clear the extension's cache (Settings > Org Management) or the browser cache if issues persist.
Export or download not working Ensure your browser allows downloads from extensions. Check download settings and security policies. Some corporate browsers block extension downloads — try a different browser profile.
Inline edit not working Verify the field is editable (not a formula, auto-number, or system field). Check that per-org inline edit safety is not blocking you (Settings > Org Management). Ensure you have write permissions on the object and field.
Platform Events or LMS not streaming Verify the channel/event exists and you have publish/subscribe permissions. Check the connection status indicator. Try refreshing the page. Streaming requires an active CometD connection — some firewalls may block long-polling.
Custom favicon not applying Click Save & Apply after designing the favicon. Make sure you have the org selected in the Saved Orgs list. The favicon applies to all open Salesforce tabs for that org — open a new tab if needed.
Keyboard shortcuts not working Other extensions or browser settings may conflict. Go to Settings > General and rebind to a different key combination. On Mac, use Cmd instead of Ctrl. Check that the extension is enabled and has permissions for the current page.
Settings lost after browser update Use Settings > Backup > Quick Export regularly to save a JSON backup of all settings. Restore with Quick Import after any data loss.
