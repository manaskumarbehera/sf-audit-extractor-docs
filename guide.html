Guide: From install to daily power use

This guide combines setup steps, practical workflows, screenshots, and a toggle-ready YouTube learning area.

Open Quick Start Open Full Docs

Recommended flow

  1. Install and validate session.
  2. Run first SOQL/GraphQL tasks.
  3. Set org-safe visual markers.
  4. Use quiz for team onboarding.

Step 1: Install and open the extension

Use Chrome Web Store install, then open a Salesforce tab so session-aware tools can initialize.

Tip: Keep one Salesforce tab open while testing query and scanner tools.

Starter command

SELECT Id, Name FROM Account LIMIT 10

Run this in SOQL Builder to verify API/session health quickly.

Step 2: Walkthrough screenshots

Install extension screen

Confirm extension install and pin settings if your team uses toolbar access.

First-run checklist

  • Open Salesforce page.
  • Launch TF button/entrypoint.
  • Run SOQL sample query.
  • Check help/privacy links resolve.

Operational best practice

Use org-specific favicon colors/labels so you can instantly distinguish PROD, UAT, and DEV tabs.

Step 3: YouTube playlist embed (toggle-ready)

Paste a playlist URL (or playlist ID), then click enable. Use hide to turn it off instantly.
